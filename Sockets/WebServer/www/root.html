You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /github -- can be used to get a list of a users repositories on github
  • /projectile -- can be used to calculate range of projectile, with params speed > 0 and angle between 0 and 90
  • /palindrome -- can be used to check if a string is a palindrome with params text=String and ignoreCase=boolean
  • /random -- returns index.html
